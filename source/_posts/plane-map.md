### 11.1 平面图的基本概念

1. **可平面图(planar graph)**：==可以==画在平面上，使得边与边不在非顶点处相交的图

2. **平面嵌入(imbedding)**：画在平面上使得边与边不在非顶点处相交的图

3. **平面图(plane graph)**：在平面上边与边==不在非顶点处相交==的图

4. 平面图：

   + $K_1,K_2,K_3,K_4,K_5-e(K_5\text{删除任意一条边})，K_{3,3}-e$

   非平面图：

   + $K_5,K_{3,3}$

5. **约当曲线**：自身不相交，且始点和终点重合的曲线

6. ==**约当定理**==：设**L**是平面Ⅱ上的一条约当曲线，平面的其余部分被分成了两个不相交的开集，分别称为**L**的内部和外部，则连接**L**的内部点和外部点的任何连续曲线必与**L**相交。

7. **球面嵌入**：画在球面上使得边与边不在非顶点处相交

8. **曲面嵌入**：画在曲面上使得边与边不在非顶点处相交，如环面嵌入

9. **定理 11.1**：可平面嵌入$\iff$ 可球面嵌入

   推论：

   + $G'$ 是平面图 G 的球面嵌入

   + $G''$ 是 平面图 G  的平面嵌入

     $\Longrightarrow$ $G' \cong G''$

10. **面和次**

    + **区域(region)**：不含顶点与边的极大连通曲面，R
    + **外部区域(exterior region)**：面积无限的区域，$R_0$
    + **区域边界(boundary of region)**：与 R 关联的边与顶点构成的子图
    + **面(face)**：区域及其边界
    + **无限面**(外部面)：面积无限的面
    + **有限面**(内部面)：面积有限的面
    + 面的**次数(degree)**：$deg(R) = \text{边界长度}$


11. **定理 11.2**：$\sum_{i=1}^rdeg(R_i) = 2m$

    > 即 图中所有面的次数之和 = 边数的两倍

12. **定理 11.3**：任何平面嵌入的内部面都可以在另一种平面嵌入下成为外部面

13. **极大平面图**：==是平面图==，但是在任意两个不相邻顶点之间加边就是非平面图。

    $(K_1,K_2,K_3,K_4,K_5-e,K_{3,3}-e)$

    特点：

    + 极大平面图一定连通
    + 极大平面图不含==割点==、==桥==

    **极小非平面图**：非平面图删除任意一条边变为平面图。如$K_5,K_{3,3}$

14. **定理 11.4**：$n(n\geq 3)\text{阶简单连通平面图是极大平面图}\iff \forall R, deg(R) = 3$

15. **定理 11.5**：

    $n(n\geq4)\text{阶极大平面图}G\text{中}, \delta(G)\geq 3$

16. **欧拉公式**：

    $\text{设}G\text{是连通平面图}\Longrightarrow n-m+r=2$

    + r ：G 的面数
    + n ：G 的阶
    + m ：G 的边数

    > **欧拉公式推广**：
    >
    > $\text{设}G\text{是连通平面图}\Rightarrow n-m+r=1+P$
    >
    > + r ：G 的面数
    > + p ：G 的连通分支数

17. **定理 11.8：**

    $\text{设}G\text{是连通平面图},G\text{的个面的次数至少是}l(\geq 3),\text{则}m\leq \frac{(n-2)l}{l-2}.$

    推广：

    $\text{设平面图}G\text{有}p\text{个连通分支},G\text{的各面的次数至少是}l(\geq 3),\text{则}m\leq \frac{l}{l-2}(n-p-1)$m.

18. **定理 11.10：**

    $\text{设}n(n\geq 3)\text{阶简单平面图}G\text{有}m\text{条边，则}m\leq 3n-6.$

19. **定理 11.11：**

    $\text{设}n(n\geq 3)\text{阶简单极大平面图}G\text{有}m\text{条边，则}m = 3n-6.$

    > ==注意：==$m=3n-6$ 是 极大平面图$(n\geq3\text{时})$的必要条件。

20. **定理 11.12：**

    $\text{设}G\text{是简单平面图，则其至少存在一个顶点，且其度数}d(G)\leq 5.$



### 11.3 平面图的判断

1. **同胚（homomorphism）$\iff$  2度结点内同构：**

   + **插入2度顶点：**把$(u,v)$变成$(u,w),(w,v)$
   + **删除2度顶点：**$d(w) = 2$，把$(u,w),(w,v)$变成$(u,v)$
   + **同胚：**反复插入或删除2度顶点后同构


2. **Kuratowski 定理**

   + **定理 11.13：**

     $\text{图}G\text{是平面图}\iff G\text{没有与}K_5\text{或}K_{3,3}\text{同胚的子图}.$

   + **定理 11.14：**

     $\text{图}G\text{是平面图}\iff G\text{没有可以收缩到}K_5\text{或}K_{3,3}\text{的子图}.$

   + >**判定平面图的算法：**
     >
     >1966，Lempel，Euler，Cederbaum，灌木生长算法

### 11.4 平面图的对偶图

1. **对偶图：**

   设 $G=<V,E>$(是平面图的某一个平面嵌入)，具有面$R_1，R_2,...,R_n$。构造图$G\text{的对偶图}G* = <V*,E*>$的方法如下：

   + 在 G 的每一个面 $R_i$ 中放置$G*$的一个顶点 $v_i*$，$V* = \{v_i*|i=1,2,...,r\}$.

   + 设 $e_k $是 G 的一条边，若$e_k$在 G 的面 $R_i$与$R_j$的公共边界上，则作边$e_k* = (v_i*,v_j*)$，且与$e_k$相交；

   + 当且仅当$e_k$只是一个面$R_i$的边界时，则作环$e_k* = (v_i*,v_j*)$并于$e_k$相交。

     $E*=\{e_k*|k=1,2,...,m\}$.

   > 1. $G*$是平面图，而且是平面嵌入。
   > 2. 若 边 $e_k$是 G 的环，则它对应的边$e*$是 $G*$的桥，若 $e_k$是G中的桥，则$e*$是$G*$的环。
   > 3. $G*$是连通的。
   > 4. 若 $G$ 的，面$R_i$与$R_j$的边界上至少有两条公共边，则关联$v_i*$与$v_j*$的边有平行边，$G*$多半是多重图。
   > 5. 同构的对偶图不一定是同构的。

2. **定理 11.15：**

   设$G*$是连通平面图G的对偶图：

   + $n = |V(G)|,n*=|V(G*)|$
   + $m = |E(G)|, m*=|E(G*)|$
   + $r\text{是}G\text{的面数}$，$r*\text{是}G*\text{的面数}$
   + $p\text{是}G\text{的连通分支数}$
     + $\therefore\quad n* = r,m*=m,r*=n-p+1$
   + 设 $G*$的顶点$v_i*$位于G的面$R_i$中，$\therefore d_{G*}(v_i*) = deg(R_i)$.

3. **对偶图的性质：**

   + 对于连通图$G，G*$与$G$互为对偶图。

   + 如果 G 的平面图，则 $G*$ 也是平面图。

   + 若边 e 是 G 中的==环==，则 $G*$与 e 对应的边 $e*$ 是 ==桥==；

     若 e 是 ==桥==，则 $G*$中与 e 对应的边 $e*$是 ==环==。

   + 在多数情况下，$G*$ 含有平行边。

   + 同构的平面图的对偶图不一定同构。

   + 任何平面图的对偶图总是连通的

4. **子对偶图：**$G\cong G*$

5. **轮图$W_n$**：在$n-1(n\geq 4)$边形$C_{n-1}$内放置一个顶点，使其与$C_{n-1}$上$n-1$个顶点均相邻。

   + 奇阶轮图$W_n$：n 是 奇数
   + 偶阶轮图$W_n$：n 是 偶数

   > $n\geq 4$ 时，轮图$W_n$是自对偶图

6. **外平面图：**

   + 定义：

     + 外平面图：平面图的所有顶点都在一个面的边界上


     + 极大外平面图：本身就是外平面图，但是在不相邻顶点之间加边就不是外平面图了。


   + 外平面图的充要条件：

     **定理 11.22：**$G\text{是外平面图}\iff G\text{不含与}K_4\text{或}K_{2,3}\text{同胚的子图}.$
     
   + 极大外平面图的充要条件：

     **定理 11.19：**

     设 G 是 所有顶点在外部面边界上的$n(\geq3)$阶外平面图，则$G$是极大外平面图$\iff$G 外部面边界是 $n-\text{圈}$，所有内部面边界是 $3-\text{圈}$。（”三角剖分“）

7. **极大外平面图的特性**

   1. **定理 11.20：**

      设 G 是所有顶点在外部面边界上的 $n(\geq3)$阶极大外平面图，则G有$n-2$个内部面。

   2. **定理 11.21：**

      设 G 是 $n(\geq 3)$阶极大外平面图$\Longrightarrow$

      + $m = 2n -3$
      + 至少有 3 个顶点度数 $\leq3$
      + 至少有 2 个顶点度数 $= 2$
      + $\kappa = 2$

### 11.6 平面图与哈密顿图(非重点)

1. **平面哈密顿图（充分条件）**

   定理($Tutte, 1956$)：4 连通平面图 是 哈密顿图。

2. **平面哈密顿图（必要条件）**

   定理($Grinberg,1968$)：设 G 是 n 阶简单平面哈密顿图，C 是其中的哈密顿回路，在 C 内（外）部次数为 $i$ 的面数为 $r_i'(r_i'')$，则
   $$
   \sum_{i=3}^n(i-2)(r_i'-r_i'') = 0
   $$
