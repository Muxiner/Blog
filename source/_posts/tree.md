### 9.1 无向树的定义及性质
**树**: 不含（简单、初级）回路的连通图。

+ **无向树**（T）: 一个连通、无回路的无向图。
+ **森林**: 每个连通分支都是树的无向树。（$p\geq2$)
+ **树叶$v$**: $d(v) = 1$。
+ **分支点$v$** $d(v) \geq 2$。
+ **平凡树**: 平凡图，既无树叶，也无分支点。

<span style="color: #ff0000; ">定理 9.1</span>
设 G = <V, E> 为 n 阶 m 条边的无向图， 则下面各命题等价：

1. G 是 树(连通无回路)
2. G 中任意两顶点之间存在惟一的一条路径
3. G 中没有圈， 且 `m = n - 1`
4. G 是连通的， 且 `m = n - 1`
5. G 是连通的， 且 G 中任何边均为`桥`
6. G 没有圈，但 G 中任二不同顶点$u,v$之间添加边$(u, v)$，所得图含惟一的一个圈

#### 无向树的性质
<span style="color: #ff0000; ">定理 9.2</span>
设Ｔ是ｎ阶非平凡的无向树，则Ｔ至少由两片树叶。

> 解释：树最少有两片树叶



证明：
设 T 有 x 片树叶，由` m = n - 1`及握手定理：
$$
2m = 2(n - 1) = \sum_{v_i \in V(T)} d(v_i) \geq x + 2(n - x) \Longrightarrow  x \geq 2
$$

### 9.2 生成树

**定义 9.2**：

+ **生成树T**：T 是 G 的生成子图，且T为树
+ T的**树枝**：$e\in E(G)且e \in E(T)$
+ T的**弦**：$e\in E(G)且e \notin E(T)$
+ T的**余数$\overline{T}$（或T的补）**:$G[E(G)-E(T)]$ 
  + 注意：$\overline{T}$不一定是树

**（生成树的存在性定理）定理 9.3**：
无向图G具有生成树$\iff$G是连通图
即
+ $\Longrightarrow$ 任何一个连通图都有生成树。
+ $\Longrightarrow$ 含有生成树的无向图是连通图。
+ 推论：
  1. G为n阶m条边的无向简单图$\Longrightarrow m \geq n - 1$.
  2. T为n阶m条边的无向连通图G的一棵生成树$\Longrightarrow$T的余树中含`m-(n-1)`条边.
  3. T是连通图G中一棵生成树，$\overline{T}$是$T$的余树，C为G中任意一图$\Longrightarrow E(\overline{T}) \bigcap E(C) \not ={\emptyset}$

**定理9.4**：？
设 T 是无向图G中的一棵生成树，e 是 T 的任意一条弦$\Longrightarrow$ $T \bigcup e$中存在只含一条弦其余边均为树枝的圈，而且不同的弦对应的圈是不同的。

<span size = 4>基本回路</span>

**定义9.2**：
设T是n阶m条边的无向连通G的一棵生成树，设

+ T 的 弦：$e_1',e_2',...,e_{m-n+1}'$， 
+ **<span style="color: #ff0000; ">$C_r$</span>:  T 添加$e_r'$产生的G中惟一的**圈(由$e_r'$和树枝组成)
+ `r = 1,2,...,m-n+1.`

则称
+ **<span style="color: #ff0000; ">$C_r$</span>** 是对应弦$e_r'$的**基本回路**或**基本圈**.

+ $\{C_1,C_2,...,C_{m-n+1}\}$是G对应T的**基本回路系统**

+ G的**圈秩**：`m-n+1`，记作：$\xi (G)$

  

**定理9.5**：
+ 设T是连通图G的一棵生成树，e为T的一条树枝$\Longrightarrow$G中存在只含树枝e，其余元素均为弦的割集。
+ 设$e_1,e_2$是T的不同的树枝$\Longrightarrow$其对应得只含一条树枝的割集是不同的。


**定义9.4**：
设
+ T是n阶连通图G的一棵生成树
+ **T的树枝**：$e_1',e_2',...,e_{n-1}'$
+ $S_i$：G的只含树枝$e_i'$的割集
+ `i = 1,2,...,n-1`

则称
+ $S_i$是G的对应生成树T由树枝$e_i$产生的**基本割集** 
+ $S_1,S_2,...,S_{n-1}$是G对应T的**基本割集系统**
+ **割集秩**：`n - 1`, 记作 $\eta(G)$

易得：
+ G中的基本割集数  = 生成树T的树枝数(边数)

<span size = 4>G的生成树的个数$\tau (G)$</span>

**定理9.6**：
$G$ 是 $n$ 阶无向连通标定图($V=\{v_1,v_2,...,v_n\}$)，则对于 $G$ 的任意非环边 $e$ 均有 
$$\tau (G) = \tau (G-e) + \tau (G\backslash e)$$
其中$G-e$是删除边，$G \backslash e$是收缩边。

**定理9.7**：
$\tau (K_n) = n^{n-2}\quad(n\geq2)$，其中$K_n$为$n$阶标定的完全图。

### 9.3 环路空间

设无向标定图$G=<V,E>$无独立顶点，$V=\{v_1,v_2,...,v_n\},E = \{e_1,e_2,...e_m\}$,$\varnothing$是G的边导出子图，则G有$2^m$个不同的边导出子图，并设为$G_1,G_2,...,G_{2^m}$，
记$$\Omega = \{G_1,G_2,...,G_{2^m}\}$$,
设$g_i = G[e_i],i=1,2,...,m$，记
$$M=\{g_1,g_2,...,g_m\}$$
有一下定理：
**定理9.8**：
$\Omega$对环合运算及数乘运算：
$$
0·G_i=\varnothing,\quad 1 · G_i = G_i,\quad i=1,2,...,2^m
$$
构成数域$F=\{0,1\}$上的n维线性空间，其M为生成元素。

**定义**：

+ <span style="color: #ff0000; ">**环路**：$G$ 中圈或若干个边不重的圈的并。记为$\varnothing$</span>
  **注意**：
  + 图中圈、回路都是环路，但环路不一定是回路，因为环路不连通 
  + $C_环$ ：G中各种环路的集合

**定理 9.9**：
设

+ T是n阶m条边的无向连通图G的一棵生成树
+ $C_k$是对应弦$e_k'$的基本回路
+ `k = 1,2,...,m - n + 1`

则 任意的r($1 \leq r \leq m-n+1$)条弦$e_1',e_2',...,e_r'$均在
$$
C_{i_1}\bigoplus C_{i_2}\bigoplus ...\bigoplus C_{i_r}
$$
其中$\bigoplus$为图之间的环和运算。

**定理 9.10**：
设$C_1、C_2$是无向图G中的任意两个回路（初级的或简单的），则 **环和 **$C_1 \bigoplus C_2$为G中的环路。
          *推论*：

+ 设$C_1、C_2$是无向图G中的两个环路，则$C_1 \bigoplus C_2$为G中环路（即 环路对环和运算时封闭的）

**定理9.11**：
设

+ G为无向连通图
+ T为G的任意一棵生成树

则G中任一回路（初级的或简单的）或为T的基本回路或为若干个基本回路的环和。

推论： 
1. 无向连通图G中任一环路或为某棵生成树的基本回路，或为若干个基本回路的环和。
2. 设G是n阶m条边的无向连通图，设G中由s个回路（初级的或简单的），则
   $$m-n+1\leq s\leq 2^{m-n+1}-1$$
3. 设G是n阶m条边的无向连通图，设s是G中环路数（含$\varnothing$),则
   $$S = 2^{m-n+1}$$



**定理9.12**：
设G为n阶m条边的无向连通图，设$G_环$为G中环路（含$\varnothing$)组成的集合，则$G_环$是$\Omega$的`n-m+1`维的子空间，其中$\Omega$是G的所有边导出子图的集合。

*注*：$|\Omega |= 2^m$

### 9.4 断集空间
**定义9.6**：
设$G=<V,E>$是一个无向图，$V_1 \subset V且V_1 \not ={\varnothing}$，记作 $\overline{V_1} = V - V_1$,则
+ G中的**断集**：$\{(u,v)|u\in V_1 \wedge v \in \overline{V_1}\}$
+ 记作 $E(V_1\times \overline{V_1})$
+ 简记 $(V_1,\overline{V_1})$
+ 割集是断集，但断集不一定是割集



**定理9.13**：
连通图G中每个割集至少包含G的每棵生成树的一个树枝。



**定理9.14**：
设
+ 设G是n阶m条边的无向连通图
+ T是G的一棵生成树
+ $S_基$是T对应的基本割集系统

则对于任意的$S_{i_1},S_{i_2},...,S_{i_k}\in S_基$，必有它们对应的树枝$e_{i_1}',e_{i_2}',...,e_{i_k}'$均在
$$
S_{i_1}\bigoplus S_{i_2}\bigoplus...\bigoplus S_{i_k}
$$
中，其中$\bigoplus$为对称差运算。



**定理9.15**：
设$S_1,S_2$为无向图G的两上断集，则$S_1\bigoplus S_2$为G中断集，其中$\bigoplus$为**对称差运算**。



**定理9.16**：
设

+ 设G是n阶m条边的无向连通图
+ T是G的任意一棵生成树

则G中任一**断集**或为T的**基本割集**或为**若干个基本割集的对称差集**。



**定理9.17**：

+ 设G为n阶m条边的无向连通图，
+ 设$S_断 = \{\varnothing\} \bigcup \{S'是G的割集的导出子集\}$

则$S_断$是$\Omega$的`n-1`维子空间，其中$\Omega$是G的所有边导出子图集合。

### 9.5 根树

+ **有向树**：基图是有向图的树。

**定义9.7**：

+ **根树**：有向树T是平凡树或T中有一个顶点的`入度 = 0`，其余顶点的入度均 `= 1` .
   + **树根**：`入度 = 0`的顶点
   + **树叶**：`入度 = 1`，`出度 = 0`的顶点
   + **内点**： `入度 = 1`，`出度 不为 0`的顶点
   + **分支点**：`内点`和`树根`
   + $v$的**层数**：从`树根`到T的任一顶点$v$的通路（路径）的长度
   + **树高**：`层数最大` 的 `顶点的层数`

**定义 9.8 家族🌳**：
设T是一棵根树，$v_i,v_j \in V(T)$

+ $v_i是v_j$的**祖先**，$v_j是v_i$的**后代**：$v_i$可达$v_j$
+ $v_i是v_j$的**父亲**，$v_j是v_i$的**儿子**：$v_i$邻接到$v_j$
+ $v_k$，$v_j$是**兄弟**：$v_k,v_j$的父亲相同

**定义9.9**： 
T为**有序树**$\Longleftarrow$T是一棵根树，将T的层数相同的顶点都标定上序号。

**定义9.10**：
设T是一棵根树：
+ T为**r叉树**$\iff$T的每个分支点至多有r个儿子
+ T为**r叉正则树**$\iff$T的每个分支点都恰好有r个儿子
+ T为**r叉完全正则树**$\iff$T是r叉正则树且每个树叶的层数 = 树高
+ T为**r叉有序树**$\iff$T是r叉树、有序树
+ T为**r叉正则有序树**$\iff$T是r叉正则树、有序树
+ T为**r叉完全正则有序树**$\iff$T是r叉完全正则树、有序树

> 2叉有序树和2叉正则有序树在数据结构中居重要地位

**定义9.11**：
设T为一棵根树，$v \in V(T)$, 称v及其后代的导出子图$T_v$为T的以v为根的**根子树**。
+ 分支点的**左子树**、**右子树**：2叉正则有序树的每个分支点的两个儿子导出的根子树

对一棵根树的每个顶点都访问且仅访问依次称为行遍一颗根树或周游一颗根树。
设T是一棵2叉正则有序树，按对树根、左子树、右子树的不同的访问顺序主要有以下3种行遍方法：
1. **中序行遍法**：先左子树、再树根、再右子树
2. **前序行遍法**：先树根、再左子树、再右子树
3. **后序行遍法**：先左子树、再右子树、再树根
