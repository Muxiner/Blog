+ 前提条件：$无向图 G = <V,E>, V^*\subseteq V$

#### 🌼 支配集

+ **支配集**：

  + $\forall u (u\in V - V^*\rightarrow\exist v(v\in V^* \wedge (u,v)\in E))$
  + $\forall u \in V-V^*, \exist v\in V^*,uFv$

  > 支配集是一个顶点集。
  >
  > 支配集$\Rightarrow$==该集合中的顶点==能够和==不在该集合的顶点==**共边**
  >
  > 寻找方法：先找度数最大的顶点，再找能够和其他顶点共边的顶点，直到这些顶点能够和其他的顶点共边。

+ **极小支配集：**<u>真子集不是支配集</u>的==支配集==。

+ **最小支配集：**顶点数最小的支配集。

+ **支配数：**==最小支配集==中的顶点个数，$\text{记作：}\gamma_0(G),\quad\text{简记：}\gamma_0$

+ **定理 13.1**

  $\text{无向图G无孤立点}，V_1^*\text{是极小支配集}，\text{则存在}V_2^*\text{也是极小支配集，且}V_1^*\cap V_2^* = \varnothing$

  > 即  ”<u>同一个图中的极小支配集之间没有相同的顶点</u>“

#### 🌼 (点)独立集

+ **(点)独立集：**

  + $\forall u,v\in V^*,(u,v)\notin E$

  > 独立集也是一个顶点集。
  >
  > 独立集$\Rightarrow$集合中的顶点之间==互不相邻==（不共边）。
  >
  > 寻找方法：直接找图中不相邻顶点。

+ **极大独立集：**（其真母集不是独立集）继续添加顶点就不再是独立集的集合。

+ **最大独立集：**最大的独立集。

+ **点独立数**：最大点独立集的顶点个数，$\text{记作：}\beta_0(G),\quad\text{简记：}\beta_0$

+ **定理 13.2**

  $\text{无向图}G\text{无孤立点，}V^*\text{是极大独立集，则}V^*\text{也是极小支配集}$

  > 逆命题不成立：即$V^*$是极小支配集，就不会是极大独立集。
  >
  > > 即 <u>可以先求极大独立集来求极小支配集，但是不能先求极小支配集，再求极大独立集</u>。
  > >
  > > $\because \text{极小支配集中的顶点可能相邻} $

#### 🌼点覆盖集

+ **点覆盖(集):**

  + $\forall e(e\in E\rightarrow\exist v(v\in V^* \wedge v\text{关联}e))$
  + $\forall e\in E, \exist v\in V^*,v\text{关联}e$

  > 点覆盖集还是一个点的集合。
  >
  > 点覆盖 $\Rightarrow$集合中的顶点是图中==所有边的端点==。
  >
  > 寻找方法：先找度数最大的顶点，然后再找其他的顶点进行补充。

+ **极小点覆盖集：**再减少顶点就不是点覆盖的集合。

+ **最小点覆盖集：**最小的点覆盖。

+ **点覆盖数：**最小点覆盖集的元素个数，$\text{记作：}\alpha_0(G),\quad\text{简记：}\alpha_0$

> + （连通图）点覆盖是支配集。
> + 极小点覆盖不一定是极小支配集。
> + 支配集不一定是点覆盖。

+ **定理 13.3**

  $\text{无向图}G\text{无孤立点，}V^*\subset V,V^*\text{是点覆盖}\iff V-V^*\text{是独立集}$

  **推论：**

  $\text{无向图}G\text{无孤立点},V^*\text{是极（最）小点覆盖}\iff V-V^*\text{是极（最）大独立集}.\text{且}\alpha_0 + \beta_0 = n$

#### 🌼团

+ **团：**$G[V^*]\text{是完全子图}$，即是完全图。

  > 团还是一个顶点的集合。
  >
  > 团$\Rightarrow$存在是完全图的子图的顶点
  >
  > 寻找方法：直接找是完全图的子图。

+ **极大团：**在加一个顶点就不是团的集合。

+ **最大团：**最大的团。

+ **团数：**最大团的顶点数。$\nu_0(G) = |V^*|$

+ **定理 13.4**

  $\text{无向图}G,V^*\text{是}G\text{的团}\iff V^*\text{是}\overline{G}\text{的独立集}$

  **推论：**

  $\text{无向图}G,V^*\text{是}G\text{的极（最）大团}\iff V^*\text{是}\overline{G}\text{的极（最）大独立集}.\color{red}\nu_0(G) = \beta_0(G).$

  > $\color{red}\alpha_0 + \beta_0 = n\text{(无孤立点)}$
  >
  > $\color{purple}\nu_0(G) = \beta_0(\overline{G})$



+ 前提条件：$G=<V,E>,E^* \subseteq E$

#### 🌼边覆盖（集）

+ **边覆盖（集）：**$\forall v\in V,\exist e\in E^*,使得 v 与 e 关联,称 e覆盖v$

  $E^*$：边覆盖集

+ **极小边覆盖集：**

+ **最小边覆盖集：**边数最少的边覆盖集

+ **边覆盖数：**$\alpha_1(G)$，简记 $\alpha_1$

#### 🌼边独立集（匹配）

+ **边独立集：**$E^*$中任何两条边均不相邻，$E*$是边独立集。

  或称：$E^*$为$G$中的**匹配**

+ **极大匹配**

+ **最大匹配**：边数最多的匹配

+ **边独立数 或 匹配数：**$\beta_1(G)$



设 M 为 G 中一个匹配：

1. 设$e = (v_1,v_j)\in M$，称$v_i,v_j$被M匹配。
2. $\forall v\in V(G),\exist e \in M,使得e与v关联，则称 v 为 M的$**饱和点**$，否则为$**非饱和点。**
3. M为**完美匹配**：G 中 每个顶点都是M的**饱和点**
4. **交错路径**: 在M和$E-M$中交替取边的路径
5. **可增广交错路径**: 两端都是==非饱和点==的**交错路径**



+ 定理 13.5：

  无向图G**无孤立点**，

  1. 设 M 是最大匹配，$\forall$非饱和点 v ，取与 v 关联的任一边，组成边集 $N$，则 $W = M \cup N$是最小边覆盖。
  2. 设 $W_1$是最小边覆盖，若$W_1$中有相邻边， 就删除其中一条边，知道无相邻边为止，设删除的边组成边集$N_1$，则$M_1 = W_1 - N_1$是最大匹配
  3. $\alpha_1 + \beta_1 = n$

+ 定理 13.6：

  无向图G**无孤立点**，

  + M 是匹配， 
  + N 是点覆盖， 
  + Y 是独立集， 
  + W 是边覆盖，

  则

  1. $|M|\leq|N|$
  2. $|Y|\leq|W|$
  3. 当 = 成立时，
     + M——最大匹配
     + N——最小点覆盖
     + Y——最大独立集
     + W——最小边覆盖

  **推论：**

  无向图G无孤立点，则$\beta_1\leq\alpha_0,\beta_0\leq\alpha_1$


#### 🌼完美匹配

+ **完美匹配(perfect matching)**: 没有非饱和点的匹配

+ **完备匹配(complete matching)** :

  $G=<V_1,V_2,E>是二部图, |V_1|≤|V_2|,|M|=|V_1|$



+ 定理 13.7

  设$M_1,M_2$是G中2个不同**匹配**,则$G[M_1⊕M_2]$的每个连通分支是$M_1,M_2$中的边组成的**交错圈**或**交错路径**

+ 定理 13.8

  设$M$是$G$中匹配, $Γ$是$M$的**可增广路径**, 则$M’=M⊕E(Γ)$也是G中匹配, 且$|M’|=|M|+1$

+ 定理 13.9$(Berge,1957)$

  $M是G中最大匹配⇔G中无M可增广路径$

+ 定理13.10$(Tutte,1947):$ 

  $G有完美匹配⇔ ∀V'⊂V(G), p_奇(G-V')≤|V'|.$

  + 说明: $p_奇$是奇数阶连通分支数
  + 推论：**无桥3正则图有完美匹配**

#### 🌼完备匹配

$G=<V_1,V_2,E>是二部图, |V_1|≤|V_2|$

+ 完备匹配: $|M|=|V_1|$

+ Hall条件: $∀S⊆V_1, |S|≤|N(S)|, 其中N(S) =\{ u | ∃v∈S,(v,u)∈E \} = U_{v∈S}Γ(v)$

+ $t(≥1)$   条件:$ ∀v∈ V_1, d(v)≥t; ∀v∈V_2, d(v)≤t$

+ $t条件⇒ ∃完备匹配⇔ Hall条件$ 

+ 定理13.11$(Hall,1935)$: 

  $二部图G有完备匹配⇔ G满足Hall条件(|S|≤|N(S)|)$

+ 定理13.12: 

  设$G=<V_1,V_2,E>$是二部图,若$V_1$中每个顶点至少关联$t(t≥1)$条边,而$V_2$中每个顶点至多关联$t$条边,则$G$中存在**完备匹配**

+ 定理13.13: 

  $设G=<V_1,V_2,E>是k正则二部图,则G中存在k个边不重的完美匹配$

+ 定理13.14:

  设$G=<V_1,V_2,E>$是无孤立点二部图,则$\alpha_0=\beta_1$
